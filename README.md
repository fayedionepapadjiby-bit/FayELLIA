# 🌟 FAYELLIA - E-commerce Boutique Feminin

**Plateforme de vente en ligne pour accessoires et vêtements féminins à Dakar, Sénégal**

---

## 📱 À propos

**FAYELLIA** est une boutique en ligne moderne et responsive spécialisée dans la vente d'accessoires et de vêtements exclusivement féminins. Conçue pour le marché sénégalais, elle intègre les méthodes de paiement locales (Wave et Orange Money) pour une expérience d'achat simple et sécurisée.

**Localisation:** Dakar, Sénégal  
**Spécialité:** Accessoires & Vêtements Feminin  
**Paiements:** Wave | Orange Money

## 🎨 Personnalisation

### Modifier les Couleurs
Ouvrez le fichier HTML et cherchez `:root` dans le CSS:
```css
--primary: #D4A574;      /* Couleur principale (or) */
--accent: #8B4789;       /* Couleur secondaire (violet) */
--dark: #2D2D2D;         /* Texte sombre */
--light: #F9F7F4;        /* Couleur de fond */
```

### Modifier le Logo
Cherchez `.logo` dans le CSS et modifiez le texte ou ajoutez une image

### Ajouter/Modifier les Informations
Dans le panneau admin, allez à **Accès & Configuration** pour mettre à jour:
- Nom de la boutique
- Email
- Téléphone
- Localisation
- Numéros de paiement
